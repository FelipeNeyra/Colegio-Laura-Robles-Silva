
1.- Nombre de empresa: Colegio Laura Robles Silva.

2.- Descripción del negocio: Como establecimiento de educación básica, la ESCUELA LAURA ROBLES SILVA se dedica 
a sentar los cimientos educativos durante la niñez, proporcionando una formación inicial integral que es crucial 
para el desarrollo futuro.

3.- Necesidad digital: Se propone desarrollar una aplicación web que permita gestionar de manera digital los 
horarios y el uso de la sala de computación del Colegio. Esta plataforma permitiría registrar, organizar y 
visualizar los horarios disponibles de forma  clara.

En el Colegio, la gestión de la sala de computación actualmente se realiza de forma manual, método que presenta 
diversas dificultades, dado que es un proceso manual. Existe una mayor posibilidad de cometer errores en la 
asignación de horarios, lo que puede generar confusiones o conflictos cuando más de un curso necesita utilizar 
la sala.
